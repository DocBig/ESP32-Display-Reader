ESP32 Display Reader

Web Installer fuer die Firmware des ESP32 Display Reader. Die Firmware liest optisch 7-Segment-Anzeigen und segmentbasierte Symbole aus und stellt die erkannten Werte ueber MQTT bereit.

Wichtig: Das Board muss PSRAM besitzen. Aktuell unterstuetzt sind z. B. ESP32-CAM (AI-Thinker kompatibel) und ESP32 WROVER.

Installation

  1. ESP32 per USB mit dem Computer verbinden
  2. Chrome, Edge oder einen kompatiblen Browser verwenden
  3. Auf Firmware installieren klicken
  4. Den passenden seriellen Port auswaehlen
  5. Nach dem Flashen das Geraet neu starten

Erste Einrichtung

Nach dem ersten Start erstellt das Geraet ein WLAN mit einer SSID nach dem Muster DisplayReader-XXXX. Nach dem Verbinden ist die Einrichtung unter http://192.168.4.1 erreichbar.

Hinweise

  • USB-Datenkabel verwenden, kein reines Ladekabel
  • Bei manchen Boards muss fuer das erste Flashen die Boot-Taste gedrueckt werden
  • Falls der Browser den Port nicht findet, USB-Verbindung trennen und erneut verbinden
