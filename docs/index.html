// 03 · Erste Schritte nach dem Flash

Was passiert als nächstes?

  1. ESP32 startet einen WLAN Access Point namens DisplayReader-XXXX
  2. Mit diesem WLAN verbinden, dann im Browser 192.168.4.1 öffnen
  3. Im Webinterface dein Heim-WLAN und MQTT konfigurieren
  4. Home Assistant erkennt den Sensor automatisch per Auto Discovery
